## 2. SEO Local — Google Business Profile & Google Local Pack

### 2.1 Création et optimisation de la fiche GBP

La fiche Google Business Profile est **le levier SEO local le plus puissant** pour un artisan électricien. En 2026, les signaux GBP représentent ~35 % des facteurs de classement dans le Local Pack (Whitespark 2026).

#### Checklist de création (critique — jour 1)

| Champ | Recommandation Elekio | Pourquoi |
|-------|----------------------|----------|
| **Nom commercial** | `Elekio` (ou `Elekio — Électricien Castres` si conforme au RCS) | Inclure le mot-clé principal **activité + ville** si le nom officiel le permet |
| **Catégorie principale** | `Électricien` (Electrician) | Spécifique, pas "Artisan" ou "Entrepreneur" |
| **Catégories secondaires** | `Installateur de bornes de recharge`, `Électricien bâtiment`, `Dépanneur électrique` | Jusqu'à 9 catégories secondaires |
| **Adresse** | Siège social RCS Castres | Google classe par proximité réelle |
| **Zone de service** | Castres + 30 km (Albi, Mazamet, Lavaur, Graulhet, Labruguière, Aussillon) | Définir les villes précises |
| **Téléphone** | Numéro local en 05.XX.XX.XX.XX | Un numéro local inspire plus confiance qu'un 09 |
| **Site web** | URL définitive du site | À créer avant le lancement |
| **Horaires** | Heures d'ouverture réelles + "Urgences 24/7" | Critère de classement important |
| **Description** | 750 caractères optimisés (voir ci-dessous) | Champ le plus sous-estimé |

## 4. SEO Technique — Core Web Vitals, Schema.org, Mobile-First

### 4.1 Core Web Vitals 2026

En 2026, les seuils CWV ont été renforcés :

| Métrique | Seuil "Good" | Ce que ça mesure |
|----------|-------------|-------------------|
| **LCP** (Largest Contentful Paint) | ≤ 2.5 secondes | Vitesse de chargement |
| **INP** (Interaction to Next Paint) | ≤ 200 ms | Interactivité (remplace FID) |
| **CLS** (Cumulative Layout Shift) | ≤ 0.1 | Stabilité visuelle |

**Recommandations pour le site Elekio :**

| Action | Impact |
|--------|--------|
| Hébergement rapide (France de préférence) | LCP |
| Images optimisées (WebP, compression, lazy loading) | LCP |
| CSS minimal et non-bloquant | LCP |
| Pas de polices Google lourdes (préférer des polices système ou subset) | LCP |
| Pas de JavaScript inutile | INP |
| Dimensions explicites pour images et vidéos | CLS |
| Design minimaliste (site vitrine = léger) | Tous |

> **Objectif mobile :** Score PageSpeed Insights > 85. Le site vitrine Elekio étant simple, cet objectif est très atteignable.

## 5. AI / LLM SEO — Optimisation pour ChatGPT, Perplexity, Gemini, AI Overviews

### 5.1 Contexte 2026

> **313 % d'augmentation** du trafic IA vers les sites PME en 2025-2026. (HubSpot)
> 31 % de la Gen Z utilise l'IA comme moteur de recherche principal. (GWI)
> Les AI Overviews de Google sont déployés dans 180+ pays. AI Mode (zéro lien bleu) arrive.

**Conséquence :** Elekio doit être visible non seulement dans les résultats Google classiques, MAIS AUSSI cité par les IA génératives.

### 5.2 Principes du GEO (Generative Engine Optimization)

Le GEO, c'est l'art d'être cité comme source dans les réponses IA.

| SEO traditionnel | GEO / AI SEO |
|-----------------|--------------|
| Classe les pages dans les SERP | Fait citer la marque dans les réponses IA |
| Priorité : mots-clés | Priorité : contexte, entités, crédibilité |
| Objectif : clics | Objectif : citations + notoriété |
| Contenu optimisé pour ranking | Contenu optimisé pour extraction |

### 5.3 Stratégie Elekio pour l'AI SEO

#### 5.3.1 Structure "Answer-First" (Réponse d'abord)

Chaque page de services doit commencer par une **réponse directe** dans les 50-80 premiers mots :

> **Exemple pour la page "Installation électrique" :**
> "Elekio réalise vos installations électriques complètes à Castres et dans tout le Tarn (81). De la construction neuve à la rénovation, nos travaux respectent la norme NF C 15-100. Devis gratuit sous 48h."

> **Exemple pour la page "Dépannage" :**
> "Vous avez une panne électrique à Castres ? Elekio intervient en urgence 7j/7 pour rétablir votre courant. Appelez-nous au [numéro] — intervention rapide garantie."

#### 5.3.2 FAQ structurées (critique pour l'AI)

Chaque page service doit inclure une section FAQ avec **questions formulées comme les utilisateurs les posent aux IA**.
